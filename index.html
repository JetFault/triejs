Example Usage

To create a simple dictionary lookup for words, just create a new trie with default options

Add words and get words by prefix through a simple interface

    
  var trie = new exports.Triejs();
  trie.addWord("Alaska", "Alaska");
  trie.getPrefix("Al");

    If you need a custom data structure to store data you can extend the trie with optional parameters

    These parameters allow you to add the ability to insert, sort, and prune data as you need

      
  var trie = new exports.Triejs({
    // customize the sorting algorithm to sort by population parameter in data object
    sort: function() {
      this.sort(function(a, b) { return b.population - a.population; }
    }
  });
  trie.addWord("Phoenix", { city: "Phoenix", state: "Arizona", population: 1476331 });
  trie.getPrefix("Ph");
